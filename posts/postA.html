How I set up my webserver

Oct 26, 2025 • notes from xisoah

I’ve been running my own webserver at xarb.me for a while — here’s a simple breakdown of the stack:

  • Ubuntu + NGINX reverse proxy
  • Systemd for service control
  • Let’s Encrypt (certbot) for SSL
  • Static HTML files served from /srv/ via symlinked releases

To deploy a new version, I just untar a release into /srv/yourproject/releases/N and update current → that’s it.
